# Security Policy

Genogram Studio is a client-only app. It stores editor state in the browser and does not intentionally send genogram data to a server.

## Reporting a Vulnerability

Please report security issues privately to the repository owner instead of opening a public issue.

Genograms can contain sensitive family, clinical, legal, or personal history data. Public issues, PRs, examples, screenshots, and fixtures must not include private or identifying genogram data.

When reporting, use synthetic sample data and describe the impact, affected browser, and steps to reproduce.
