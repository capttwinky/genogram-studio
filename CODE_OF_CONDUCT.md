# Code of Conduct

Be respectful, practical, and specific. Discuss code and project decisions without personal attacks or harassment.

## Project Privacy Standard

Genograms can contain sensitive family, clinical, legal, or personal history data. Public issues, PRs, examples, screenshots, and fixtures must not include private or identifying genogram data.

Use fictional examples when discussing schema, rendering, or validation behavior.

## Enforcement

Maintainers may edit, hide, or remove content that violates these expectations, especially content that exposes private genogram data.
