# Contributing

Thanks for improving Genogram Studio. Keep changes focused and aligned with the current React, TypeScript, Vite, D3, Zod, and plain CSS setup.

## Local Checks

```bash
npm install
npm run build
```

## Privacy

Genograms can contain sensitive family, clinical, legal, or personal history data. Public issues, PRs, examples, screenshots, and fixtures must not include private or identifying genogram data.

Use synthetic examples with fictional names and dates.

## Pull Requests

- Describe the user-facing behavior changed.
- Include validation or rendering implications when schema behavior changes.
- Do not add a backend, database, authentication, or committed `dist/` output.
